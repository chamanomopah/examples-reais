# Examples Réais

Exemplos e scripts relacionados a ComfyUI, TTS (Text-to-Speech) e workflows de IA.

## Conteúdo

- ComfyUI TTS integrations (Kokoro, Qwen)
- LTX Video scripts
- Utility functions
- Workflow JSON files
- Python examples

## Scripts

- `comfyui_kokoro_tts.py` - Integração Kokoro TTS para ComfyUI
- `comfyui_ltx_video.py` - Script LTX Video para ComfyUI
- `comfyui_qwen_tts_*.py` - Variações Qwen TTS
- `kokoro_tts_exemplo1_python_simple.py` - Exemplo simples Kokoro TTS
- `zimage_turbo_exemplo1_python_simple.py` - Exemplo Z-Image Turbo
- `utils.py` - Funções utilitárias
